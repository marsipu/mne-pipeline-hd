mne_pipeline_hd.gui.parameter_widgets.SliderGui#

class mne_pipeline_hd.gui.parameter_widgets.SliderGui(min_val=0, max_val=100, step=1, tracking=True, **kwargs)#

A GUI to show a slider for Int/Float-Parameters

__init__(min_val=0, max_val=100, step=1, tracking=True, **kwargs)#
Parameters:

  • min_val (int | float) – Set the minimumx value, defaults to 0.

  • max_val (int | float) – Set the maximum value, defaults to 100..

  • step (int | float) – Set the step-size, defaults to 1.

  • tracking (bool) – Set True if values should be updated constantly while the slider is dragged (can cause crashes when heavyweight functions are connected to the ParamChanged-Signal).

  • **kwargs – All the parameters fo :method:`~Param.__init__` go here.
